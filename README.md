# 🧮 Calculator App

A simple and responsive Calculator application built using **HTML, CSS and JavaScript**. This project performs basic arithmetic operations with a clean and user-friendly interface, making it a great beginner-friendly project for learning DOM manipulation and event handling in JavaScript.

## 🚀 Features

* Addition, Subtraction, Multiplication and Division
* Clear (C) functionality
* Delete/Backspace functionality
* Responsive design for different screen sizes
* Real-time display updates
* User-friendly interface

## 🛠️ Technologies Used

* **HTML5** – Structure of the calculator
* **CSS3** – Styling and layout
* **JavaScript (ES6)** – Calculator logic and interactions

## ⚙️ How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/Bineth-Tharana/Calculator.git
```

2. Navigate to the project folder:

```bash
cd Calculator
```

3. Open `index.html` in your browser.
